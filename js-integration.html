Eventos

script.js 
<!-- no HTML -->
<button onclick="cumprimentar()">Clique aqui</button>

function cumprimentar() {
  alert("Boa tarde!");
}
  • onclick — atributo do HTML que chama uma função JavaScript quando o botão é clicado
  • function — define um bloco de código reutilizável com um nome
  • alert() — abre uma janela de aviso no navegador

14

Interação visual

script.js 
const box = document.querySelector(".box");

box.addEventListener("mouseover", () => {
  box.style.backgroundColor = "blue";
});
  • mouseover — dispara quando o mouse entra no elemento
  • style — permite alterar CSS diretamente via JavaScript
